git clean - Աշխատանքային դիրեկտորիայից մաքրում է "աղբը", որը կարող է լինել ծրագրի ստեղծման արդյունքում։

git grep ֊ Փնտրում է ցանկացած տող, ձեր ծրագրում։


git remote ֊ Ծառայում էորպես հեռակա ռեպօզիտորիաների համար։ Այն թույլ է տալիս պահել երկար ռեպո․ ֊ի URL֊ը կարճ, հասկանալի ձևով <origin> ի միջոցով։
